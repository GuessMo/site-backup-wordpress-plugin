Export and import WordPress posts (including media) between instances with post type mapping.

== Description ==

Site Backup lets you export any registered post type — including custom post types — as a ZIP archive
containing a JSON manifest and all attached media files. The archive can then be imported into another
WordPress instance, with optional post type remapping (e.g. a custom "article" type → standard "post")
and configurable collision handling (skip or override).
